Tech Stack

This website uses HTML, CSS and JS. It's deployed through Cloudflare Pages. I serve the site locally with vercel/serve. GitHub is used for "Version Control". Their mobile app is often used for quick updates. You can also use GitHub to browse the source code. The site is mainly written in VScode.

I wanted to keep it simple, my only need is to serve static content. Handcrafted pages load fast. It's easy to deploy anywhere. Why settle for complex, when it can be simple.
